## 1. 快速上手（5 分钟）

**目标**：把一份散点数据网格化，看到第一张热力图，并存成文件。

| 步 | 操作 | 说明 |
|---|---|---|
| 1 | MATLAB 中运行 `startApp` | 首次需激活授权（见第 2 节），之后直接进主界面 |
| 2 | 左侧功能树点 **网格化 → 散点插值(网格化)** | 中间输入区自动切换为「散点」形态 |
| 3 | 顶部单/批开关选 **单文件** | 处理一份散点文件 |
| 4 | 点 **输入文件** 的「浏览」，选 `Data/scatter_comma.csv` | 支持 `.txt/.csv/.dat/.xlsx` |
| 5 | 点 **输出文件** 的「浏览」，起个名如 `out.asc` | 输出格式由顶部「输出格式」下拉决定 |
| 6 | 参数区：`method` 选 `linear`，`cellsize` 填 `0`（自动） | 0 = 自动估算网格间距 |
| 7 | 点 **预览首个** | 右侧预览区出现热力图 |
| 8 | 换个配色（如 `parula`）观察即时变化 | 19 种 colormap 可选 |
| 9 | 点 **▷ 运行** | 生成 `out.asc`，日志区提示完成 |

> 你填的所有值和路径会自动保存，下次打开自动恢复（见 [4.9](#49-跨会话状态持久化)）。

## 4. 逐步操作指南

### 4.0 通用六步

1. **选功能**：左侧树点具体功能 → 中间输入区/参数区按其输入模式重建。
2. **配输入**：按输入区提示选文件或文件夹（+通配符）。
3. **配参数**：填数值 / 选下拉 / 选辅助文件（如 `.bln`、层面网格）。
4. **配输出**：底部选「输出格式」「配色」，可选「按类别分文件夹」。
5. **预览**：点 **预览首个** 只计算不写盘，右侧看效果、调参数。
6. **运行**：点 **▷ 运行** 写盘，日志区看进度与结果。

不同功能的输入区形态分四类，下面分别说明。

### 4.1 单文件 grid 模式（1 网格 → 1 网格）

适用：高斯平滑、去趋势、向上延拓、求导、FK 滤波、场分离、扩边、旋转、白化…

1. 顶部单/批开关选 **单文件**。
2. **输入文件**：浏览选 1 个网格文件（`.asc/.grd/.mat`）。
3. **输出文件**：浏览指定输出路径（扩展名跟随「输出格式」下拉）。
4. 填参数 → 预览 → 运行。输出 1 个文件。

### 4.2 批量 grid 模式（文件夹 → 文件夹）

适用：批量格式转换、批量扩边、批量延拓…

1. 单/批开关选 **批量**。
2. **输入文件夹**：选含网格文件的目录。
3. **通配符**：如 `*.asc`（只处理匹配文件）。
4. **输出文件夹**：选输出目录（不存在会自动创建）。
5. 填参数 → 预览（只算文件夹里第一个）→ 运行。
6. 日志逐文件报告 `ok/failed/skipped`；单个失败不中断整批。

### 4.3 散点网格化 scatter 模式

适用：散点插值(网格化)。输入区与 grid 相同，但输入读取器为散点读取器。

- **单文件**：选 1 个散点文件（`.txt/.csv/.dat/.xlsx`，前三列 `X Y Z`，自动跳表头）。
- **批量**：文件夹 + 通配符（如 `*.csv`），逐个网格化。
- 关键参数：`method`（14 法，见速查卡）、`cellsize`（0=自动）、`radius`（Surfer 搜索半径，0=默认）。

### 4.4 双输入 dualGrid 模式（2 网格 → 1~2 输出）

适用：时移求差、谱范数匹配、标准化到参考。**无单/批开关按钮**，但同样支持单/批：

- **单文件**：分别选 **A 文件** 与 **B 文件**。
- **批量**：分别选 **文件夹 A** 与 **文件夹 B** + 通配符；两文件夹中**同名文件自动配对**。
- 约定：差值/标准化以 **B 相对 A**（如时移求差输出 `B - A`）。标准化到参考额外输出差值网格。

### 4.5 层堆栈 stack 模式（N 网格 → 1 或 N-1 输出）

适用：层差、沿层提取、层段叠加。天然是文件夹级：

1. **输入文件夹** + **通配符**：一组不同深度的切片。
2. 深度自动从**文件名**解析（支持 `_500m`、`delt1000to2000`、`delt500`），按深度排序。
3. 部分功能需**层面文件**：
   - 沿层提取：1 个层面网格（目标深度面）。
   - 层段叠加：**顶层面** + **底层面** 两个网格。
4. 输出：层差出 N-1 张；沿层提取/层段叠加出 1 张。

> 深度缩放因子默认 2.5（`+core/parseDepth.m` 可调）。文件命名务必含可解析的深度标记。

### 4.6 预览区操作

- **热力图**（多数算子）：随「配色」下拉即时换色。
- **折线图**：径向功率谱输出 `ln(P)–k` 曲线（另存 `.csv`）。
- **直方图 + 拟合曲线**：GRD 阈值分析显示直方图、拟合线与检测到的锚点。
- **多输出切换**：多尺度分离、层差、标准化到参考等产生多个结果时，顶部 **预览文件下拉** 出现，逐项切换查看；单输出时下拉隐藏。

### 4.7 输出格式与文件命名

| 输出格式下拉 | 扩展名 | 内容 |
|---|---|---|
| Esri ASCII | `.asc` | ArcInfo 文本网格，行序北→南 |
| Surfer DSAA | `.grd` | Surfer 6 文本网格，行序南→北 |
| MAT 结构 | `.mat` | 网格结构体变量 `grid` |
| XYZ 三列 | `.xyz` | `X Y Z` 文本，NaN 跳过 |

多输出算子在扩展名前自动插入语义后缀，例如：
`_regional_s5`（区域场，尺度 5）、`_local_s5`（局部场）、`_layerdiff_1_2`（层差）、`_normalized_to_ref` / `_diff_to_ref`、`_upward`、`_enhanced`、`_fkfiltered`、`_derivative`、`_integrated`、`_normalized`、`_radial_spectrum` 等。

### 4.8 按类别分文件夹

勾选底部 **☑ 按类别分文件夹** 后，输出在目标目录下自动按功能分类建子文件夹（如 `场分离/`、`频谱/`、`网格工具/`）。不勾选则全部平铺在输出目录。

### 4.9 跨会话状态持久化

所有输入框的值、加载/保存路径、输出格式、配色等在退出时自动写入 `appState.json`，下次启动自动恢复上次工作状态，无需重复填写。

## 5. 输入数据准备规范

**散点文件**（网格化输入）：
- 纯文本前三列 `X Y Z`，分隔符可为空格、Tab 或逗号（自动识别）。
- 允许有表头行（自动跳过非数值行）。
- 支持 `.txt / .csv / .dat / .xlsx`。示例见 `Data/scatter_*.*`。

**网格文件**（多数算子输入）：
- `.asc`（Esri）、`.grd`（Surfer DSAA 文本）、`.mat`（本平台结构体）。
- 空白值：`.asc` 用 `NODATA_value`，`.grd` 用 `1.70141e38`，读入统一转 `NaN`。

**层堆栈切片**：文件名须含深度标记（`_500m` / `delt1000to2000` / `delt500`），否则无法排序。

**辅助文件**：
- 白化裁剪：`.bln` 边界多边形。
- 七参数坐标变换：`Par.txt`（示例见 `Data/Par.txt`）。
- 沿层/层段：层面网格文件（`.asc/.grd`，值为深度）。

**坐标约定**：`G.z` 第 1 行 = 最北（最高 Y）；`xll/yll` 为左下角节点坐标；最北 Y = `yll + (nrows-1)·cellsize`。

## 6. 典型场景实操

### 场景 A：散点 → 网格 → 位场增强 → 深度反演

1. **网格化**：散点插值 → `kriging(Surfer)` 或 `linear`，`cellsize` 依测网密度。
2. **高斯平滑**：`sigma=1~2` 去噪。
3. **向上延拓**：`height=500/1000/2000` 多高度，压制浅源。
4. **位场增强**：`method=水平总梯度THD` 或 `倾斜角Tilt`，勾画构造边界。
5. **欧拉反褶积**：`si=1`（岩墙）或 `2`（球体），`window=10`，反演场源深度。
6. **FK 滤波**：按目标波长设 `klow/khigh` 分离波长成分。

**判读**：THD/Tilt 零线或脊线对应异常边界；欧拉解聚集处为场源深度。

### 场景 B：多尺度场分离（配谱估深）

1. **径向功率谱**：`nBins=50`，看 `ln(P)–k` 曲线拐点估不同深度场源。
2. **多尺度分离**：
   - `method=高斯差分DoG`（或向上延拓）。
   - `scales1=2,5`（第一组逐层剥离，深度递增）。
   - `scales2=10,20`（第二组以最深区域场为基准并层剥离）。
   - `output=两者`，得每尺度的区域场 + 局部场（多输出）。
3. 预览下拉逐项对比，识别目标尺度的局部异常。

> 大网格开 `parallel=是` 提速（需 Parallel Toolbox）；超 `maxMemGB` 自动跳过并提示。

### 场景 C：时移求差（4D 监测）

1. 功能树选 **网格运算 → 时移求差(B-A)**（双输入模式）。
2. **单文件**：A=基线期网格，B=监测期网格；或**批量**：两期文件夹按名配对。
3. 运行输出 `B - A` 差值网格，反映期间场变化。
4. 若两期整体幅度有系统差异，先用 **谱范数匹配** 求最优缩放 α 再求差。

### 场景 D：层堆栈沿层提取

1. 准备切片：命名含深度，如 `bouguer_500m.asc`、`bouguer_1000m.asc`…
2. 功能树选 **沿层 → 沿层提取**。
3. 输入切片文件夹 + 通配符；选 **层面文件**（目标深度面网格）。
4. 运行 → 沿层面在每格点垂向线性插值，输出层面上的场值。
5. 区间平均改用 **层段叠加**（输入顶/底层面，选厚度平均或加权平均）。

## 7. 参数速查卡

| 功能 | 关键参数 | 推荐起点 |
|---|---|---|
| 散点插值 | method / cellsize / radius | `kriging` / `0`(自动) / `0` |
| 高斯平滑 | sigma | `2`（越大越平滑） |
| 去趋势 | order | `1`（平面）/`2`（曲面） |
| 向上延拓 | height(m) | `500~2000` |
| 垂向导数 | order | `1`（边界）/`2`（锐化） |
| 位场增强 | method / inc / dec | `THD`；化极填 `inc/dec` |
| 欧拉反褶积 | si / window / maxdep | `1` / `10` / `0`(自动) |
| 场分离 | method / scale / output | `DoG` / `5` / `区域场` |
| 多尺度分离 | scales1 / scales2 | `2,5` / `10,20` |
| FK 滤波 | klow / khigh / taper(rad/m) | 按目标 λ；`taper=0.2` |
| 径向功率谱 | nBins | `50` |
| 归一化 | method / outMin / outMax | `线性` / `-1` / `1` |
| 网格扩边 | method / extendMeters | `对称` / `5000` |
| 网格旋转 | angle(°) | 正=逆时针 |

> 波长-波数换算：`k(rad/m) = 2π / λ(m)`。例：`λ=1000m → k≈0.0063`。完整参数含义见《用户手册》§4。

## 8. 术语表

**地球物理**
- **位场 (Potential Field)**：满足 Laplace 方程的重力/磁场，可做延拓、求导等频域运算。
- **区域场 / 局部场**：低频深部平缓成分 / 高频浅源起伏成分；`局部场 = 实测 - 区域场`。
- **向上/向下延拓**：把观测面数据换算到更高/更低平面；向上压制浅源，向下增强浅源但放大噪声。
- **化极 RTP**：把斜磁化异常转成垂直磁化，使异常中心对准场源；低纬度不稳定。
- **THD / 解析信号 AS / 倾斜角 Tilt**：勾画异常边界的增强量；Tilt 不依赖场源深度。
- **欧拉反褶积**：由场及其梯度自动反演场源位置/深度；构造指数 N 表征场源几何。
- **Spector-Grant 功率谱**：`ln(P)–k` 斜率 `-2h` 估场源平均深度。

**信号 / 数值**
- **波数 k**：空间频率，本平台 `|k|` 已含 `2π`（rad/m）。
- **FFT / 频率域算子**：延拓、求导、FK 滤波、谱分析在频域完成，NaN 以均值填充参与、输出恢复。
- **锥度 taper**：滤波过渡带余弦平滑，抑制 Gibbs 振铃。
- **锚点 (Anchor)**：归一化分段线性映射的断点，可自动检测或手填。
- **cellsize / 节点配准**：网格间距；`xll/yll` 为角点节点坐标。

**平台专有**
- **输入模式**：`grid / scatter / dualGrid / stack` 四类，决定输入区形态。
- **多输出**：一次运行产生多个结果文件，预览下拉切换查看。
- **NODATA / 空白值**：无效格标记，读入统一为 `NaN`。

## 9. 排错手册

| 现象 | 可能原因 | 对策 |
|---|---|---|
| 启动提示"未找到授权模块 LicenseManager" | 授权模块不在路径 | 确认 `E:\OneDrive - mails.ucas.ac.cn\科研工作\Claude_Code\验证授权系统\LicenseManager.m` 在路径上 |
| 启动提示"输入参数太多" | 路径存在多个 `LicenseManager.m` 副本 | 移出旧副本，运行 `rehash toolboxcache` |
| 提示工具箱缺失 | 未装 Wavelet / Parallel | 避开 DWT/DTCWT 分离法；关闭多尺度并行开关 |
| 散点文件读不进 | 列数不足 3 / 全表头 | 确保前三列 `X Y Z` 有数值行 |
| 层堆栈排序错乱或报错 | 文件名无可解析深度 | 命名加 `_500m` / `delt…to…` 标记 |
| 双输入批量只出部分结果 | 两文件夹文件名不同名 | A/B 同名才配对，核对命名 |
| Surfer 插值不工作 | 未装 Surfer / COM 未注册 | 改用 MATLAB 内置法，或重装 Surfer |
| 化极结果发散 | 低纬度 `|I|<20°` | 化极在低纬本就不稳定，改用 THD/Tilt |
| 多尺度分离被跳过 | 超 `maxMemGB` 内存警戒 | 调大 `maxMemGB` 或缩小网格/尺度 |
| 日志区中文乱码 | 控制台 GBK vs 源码 UTF-8 | 不影响结果；`feature('DefaultCharacterSet','UTF-8')` |
| 预览无图 | 该功能为分析类（谱/阈值） | 看折线/直方图而非热力图 |

## 10. 最佳实践与性能建议

- **先预览后运行**：批量前用「预览首个」在单文件上确认参数，避免整批返工。
- **cellsize 别贪细**：过密网格会成倍增加内存与时间；散点用 `0`(自动) 起步再微调。
- **频域算子先补边**：强梯度数据先「网格扩边（对称）」再延拓/求导，减轻 FFT 边界效应。
- **场分离配谱走**：先「径向功率谱」估深，再据拐点设多尺度尺度组，比盲试高效。
- **大网格并行**：多尺度分离开 `parallel=是`（需 Parallel Toolbox）；注意 `maxMemGB` 拦截。
- **输出规整**：多算子混跑时勾「按类别分文件夹」，配语义后缀便于回溯。
- **善用持久化**：参数会自动记忆，重复实验只改动关键项即可。
- **NaN 心里有数**：频域算子对 NaN 做均值填充参与、输出恢复；空间域算子逐元素跳过 NaN。
