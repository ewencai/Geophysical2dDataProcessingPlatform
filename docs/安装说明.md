# 二维数据批处理平台 — 安装说明

![splash](splash.png)

**版本 V1.0.1**

## 系统要求

| 项目 | 最低要求 | 推荐配置 |
|---|---|---|
| 操作系统 | Windows 10 (64-bit) | Windows 11 (64-bit) |
| MATLAB | R2020b | R2024b+ |
| 内存 | 8 GB | 32 GB（处理 1000×1000+ 网格） |
| 硬盘 | 500 MB 可用空间 | SSD，1 GB+ |
| 显示器 | 1366×768 | 1920×1080 |

## MATLAB 工具箱依赖

| 工具箱 | 用途 | 可选性 |
|---|---|---|
| MATLAB | 核心数值计算、FFT、图形 | **必需** |
| Image Processing Toolbox | `imgaussfilt`, `imdilate`, `imerode`, `imfilter` | **必需** |
| Wavelet Toolbox | DWT (`wavedec2`/`waverec2`) 和 DTCWT (`dddtree2`) 分离方法 | 可选 |
| Parallel Computing Toolbox | 多尺度分离 `parfor` 并行加速 | 可选 |

> 未安装 Wavelet Toolbox 时，小波和双树复小波分离方法不可用。未安装 Parallel Computing Toolbox 时，多尺度分离自动回退串行模式，功能不受影响仅速度降低。

## 第三方软件依赖

| 软件 | 用途 | 可选性 |
|---|---|---|
| **Golden Software Surfer** (v13+) | 克里金、反距离加权、最小曲率、径向基函数、改进谢别德、自然邻点、三角网、移动平均、局部多项式等 9 种高级插值方法 | 可选 |
| **Surfer** (v8+) | Surfer 6 二进制网格 (.grd) 读写 | 可选 |

无 Surfer 时仍可使用 MATLAB 内置的 linear、natural、nearest、cubic、v4 插值方法。

### 4. 授权激活

**首次启动需要激活授权：**

1. 在 MATLAB 中运行 `startApp`
2. 弹出激活对话框，复制显示的「机器码」
3. 将机器码发送给开发者（`caiyun17@mails.ucas.ac.cn`）获取授权码
4. 收到授权码（`License.key` 文件内容）后，粘贴到激活框，点击「激活」
5. 激活成功后，授权文件保存至 `%APPDATA%\AppLicenseSystem\License.key`，后续启动无需重复激活

**授权技术说明：**

- 采用 RSA-2048 非对称签名（SHA256withRSA），私钥仅开发者持有
- 授权码绑定机器指纹（MachineGuid + 主板序列号 SHA-256），不可跨设备使用
- 含到期日签名，离线可验
- 高水位防回拨时钟：维护历史最大时间戳（base64 混淆），阻止通过回拨系统时间延长授权

**诊断授权状态：**

```matlab
diagLicense
```

## 支持的网格文件格式

### 输入格式

| 格式 | 扩展名 | 读取器 | 说明 |
|---|---|---|---|
| Esri ASCII Grid | `.asc` | `core.gridRead` | 文本格式，头信息+ncols×nrows 矩阵 |
| Surfer 6 Binary | `.grd` | `core.gridRead` | 二进制，单精度浮点 |
| Surfer 7 Binary | `.grd` | `core.surfer7Read` | 二进制，双精度浮点 |
| MATLAB MAT | `.mat` | `core.gridRead` | 变量 `z/xll/yll/cellsize/nodata` |
| XYZ 文本 | `.xyz` `.dat` | `core.scatterRead` | 三列 `X Y Z`，自动检测分隔符 |
| CSV | `.csv` | `core.scatterRead` | 逗号分隔，自动跳过标题行 |
| Excel | `.xlsx` | `core.scatterRead` | 三列数据 |

### 输出格式

顶部「输出格式」下拉框可选：Esri ASCII (.asc)、Surfer DSAA (.grd)、MAT 结构 (.mat)、XYZ 三列 (.xyz)。均为文本格式（`.mat` 除外）；当前版本不写 Surfer 二进制网格。

> Esri ASCII Grid 格式说明：
> ```
> ncols         <列数>
> nrows         <行数>
> xllcorner     <左下角X>
> yllcorner     <左下角Y>
> cellsize      <网格间距>
> NODATA_value  <空白值>
> <nrows 行 × ncols 列浮点矩阵>
> ```
> 行 1 = 最北（最高 Y），行 nrows = 最南（最低 Y）

## 常见问题

**Q: 启动提示"未找到授权模块 LicenseManager"**

```matlab
which LicenseManager -all
```

确认 `E:\OneDrive - mails.ucas.ac.cn\科研工作\Claude_Code\验证授权系统\LicenseManager.m` 在路径上。

**Q: 启动提示"输入参数太多"**

检查路径中是否存在多个 `LicenseManager.m` 副本。旧版本位于 `E:\OneDrive - mails.ucas.ac.cn\科研工作\Claude_Code\0.代码备份\`，应将其移出 MATLAB 搜索路径。然后运行：

```matlab
rehash toolboxcache
```

**Q: 提示工具箱缺失**

运行 `ver` 查看已安装工具箱。无 Wavelet Toolbox 时避免使用 DWT/DTCWT 分离方法；无 Parallel Computing Toolbox 时关闭多尺度分离的并行开关。

**Q: Surfer 插值不工作**

确认 Surfer 已安装并注册了 COM 组件。在 MATLAB 中测试：

```matlab
try
    app = actxserver('Surfer.Application');
    disp(app.Version);
    app.Quit();
catch ME
    disp(['COM 创建失败: ', ME.message]);
    disp('请重新安装 Surfer 或使用 MATLAB 内置插值方法');
end
```

**Q: 日志区出现乱码**

MATLAB 控制台使用系统默认编码（Windows 中文环境为 GBK），而源代码为 UTF-8。这不影响数据处理结果，仅影响日志显示。如需修正，在 MATLAB 中设置：

```matlab
feature('DefaultCharacterSet', 'UTF-8');
```
