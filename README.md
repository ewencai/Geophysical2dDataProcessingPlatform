# Geophysical 2D Data Processing Platform

2D geophysical data processing and visualization platform for potential field data (gravity and magnetics).

## Features

- 2D gravity/magnetic data processing
- Gridding and interpolation
- Frequency-domain filtering
- Derivative and continuation
- Interactive visualization

## System Requirements

- Windows 10/11 (64-bit)
- MATLAB Runtime R2026a
